# Детектор хвостов (Tail Detection App)

Приложение для обнаружения хвостов на фотографиях котов с использованием YOLO.

##  Установка

##  Клонируйте репозиторий:
```bash
git clone https://github.com/Machine-Magician/tail_detection_app.git
cd tail_detection_app


##  Установите зависимости:

python -m pip install -r requirements.txt
или
py -3.11 -m pip install -r requirements.txt     # зависимости работают на этой версии python

##  Запуск

py -3.11 app.py   # для Python 3.11 (рекомендуется)

После запуска откройте браузер и перейдите по адресу:  
**http://127.0.0.1:7861**

##  Как пользоваться

1. Загрузите фото кота через интерфейс
2. При необходимости отрегулируйте порог уверенности ползунком
3. Нажмите "Найти хвост"
4. Результат появится справа с красной маской и контуром

##  Порог уверенности
- **0.35** — стандартное значение (находит большинство хвостов)
- **0.35-0.50** — более строгий отбор
- **0.05** — максимальная чувствительность

##  Примечание
Приложение проверено на **Python 3.11**. На других версиях могут потребоваться корректировки зависимостей.
